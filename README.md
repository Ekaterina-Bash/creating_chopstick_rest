# creating_chopstick_rest
This is a description of chopstick rest creating. This process includes using epoxy resin, silicone and 3D-technologies.
# **Создание подставки для палочек суши**

Необходимо создать 
## 1. Проектирование в Rhino формы для заливки молда

шаг1. Моделирование детали в Rhino
При проектировании подставки для палочек суши удобнее всего использовать технологию SubD, которая позволяет сглаживать объекты и придавать им произвольную форму. В данном случае модель будет исполнена в форме наподобие листка дерева. Необходимо определиться с подходящими габаритами, а такюе избежать сильной детализированности и тонких элементов.

шаг2. Моделирование формы для заливки силикона

Необходимо полученную первым шагом деталь разделить на две части и расположить в двух половинках некой коробки, как указано на рисунке ниже. По завершении данного шага, мы имеем две "коробки" с нашей фигурой на дне. Также, необходимо сделать 3 цилиндра от верхней поверхности фигуры, они нужны для заливки эпоксидной смолы в силиконовую форму на последующем этапе.

## 2. Печать формы

Печать осуществляется с помомщью PLA-пластика, а нарезание на слои происходит в OrcaSlicer.

## 3. Получение молда

Первым делом, необходимо в Rhino рассчитать объем полученной полученной напечатанной формы, чтобы узнать необходимый объем силикона для заливки. Далее, в напечатанную форму заливается силикон, полученный смешиванием компонентов по инструкции. Заливаем емкость до краев. Ждем около 24 часов застывания.

<img width="320" height="240" alt="image" src="https://github.com/user-attachments/assets/fbf619cd-7cd6-4a0a-9d4f-1d93ffcfcdd8" />

## 4. Заливка молда эпоксидной смолой

После застывания молда, снимаем его и заливаем эпоксидную смолу, полученную также путем смешивания эпоксидной смолы и отвердителя. По деланию, можно добавить краситель. В данном случае используется белый краситель.

<img width="270" height="360" alt="image" src="https://github.com/user-attachments/assets/bbbc0092-bb7c-46b6-98a8-5b2cdd5997e3" />


## 5. Результат

На выходе мы получаем готовый держатель палочек суши, а силиконовую форму можно использовать повторно. Ненужные цилиндры можно отрезать кусачками, а саму деталь - доработать, механически убрав шероховатости.
