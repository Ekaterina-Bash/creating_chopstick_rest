## 1. Проектирование в Rhino формы для заливки молда

шаг1. Моделирование детали в Rhino
При проектировании подставки для палочек суши удобнее всего использовать технологию SubD, которая позволяет сглаживать объекты и придавать им произвольную форму. В данном случае модель будет исполнена в форме наподобие листка дерева. Необходимо определиться с подходящими габаритами, а такюе избежать сильной детализированности и тонких элементов.
